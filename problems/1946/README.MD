# 신입 사원

어떤 회사의 채용 심사에 N 명의 지원자들이 있고, 각 지원자들마다 서류 점수 순위, 면접 점수 순위가 있다.

회사에 채용되기 위해서는 다른 지원자보다 서류 점수 순위든 면접 점수 순위든 하나라도 우위에 있으면 된다.

아래의 표와 같은 평가를 받은 지원자들이 있다.

### CASE 1

| 지원자 | 서류 점수 순위 | 면접 점수 순위 |
| --- | --- | --- | 
| A | 1 | 1 |  
| B | 2 | 2 |  
| C | 3 | 3 | 




### CASE 2

| 지원자 | 서류 점수 순위 | 면접 점수 순위 |
| --- | --- | --- | 
| A | 1 | 2 |  
| B | 2 | 1 |  
| C | 3 | 3 | 

### CASE 3

| 지원자 | 서류 점수 순위 | 면접 점수 순위 |
| --- | --- | --- | 
| A | 1 | 3 |  
| B | 2 | 1 |  
| C | 3 | 2 |
