# 쉬운 계단 수

전혀 쉽지 않은 계단 수...

DP 로 풀리는 규칙성을 찾는데는 오래 걸리지 않았고 테스트 케이스 또한 모두 통과하였으나...

제출에서 계속 틀렸다...

원인은 sum 을 구할 때 dp 값을 mod 로 나눈 나머지 값을 더해주면 그 값이 sum 을 % mod 한 값과 같다고 생각한 나의 실수.

sum 또 mod 로 한 번더 잘 나누어 주어야 한다.

```cpp
 long long sum = 0;
    for(int i = 1; i <= 9; i++)
        sum += dp[N][i]% mod;
    cout << sum% mod;
```
